# Documentação Inicial do Projeto

Este repositório tem como objetivo realizar o controle e versionamento dos artefatos do projeto de software relacionado ao controle da produção de café.

## Conteúdos previstos neste repositório
- Documentos de requisitos
- Diagramas
- Código-fonte
- Relatórios de testes
- Artefatos da gestão ágil
- Evidências de decisões técnicas

## Objetivo do Repositório
Garantir segurança, rastreabilidade, organização e colaboração eficiente entre os membros da equipe durante todo o ciclo de desenvolvimento.
